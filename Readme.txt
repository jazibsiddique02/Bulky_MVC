Created a Book Store Web App using MVC.

The App is built using ASP.NET Core MVC and Entity Framework Core, ustilizing N-tier Architecture. It includes features such as:
// - Book management (CRUD operations)
// - User authentication and authorization
// - Shopping cart functionality
// - Order processing
// - Admin dashboard for managing books,orders,users, and companies.
// The project is structured with a clear separation of concerns, making it easy to maintain and extend.
// The application uses a SQL Server database for data storage and implements repository pattern for data access.
// Stripe is integrated for payment processing, allowing users to securely purchase books.
// Registered Companies can place orders and pay within 30 days.
// A product(book) can have multiple Images.
// Order and payment management functionality is also implemented forn Admin.
